# i3-dotfiles-minimal
Minimalistic dotfiles for i3wm

Screenshot:

![Screenshot](screenshots/fetch-only.png)

## Dependencies

* i3wm
* i3blocks
* kitty
* rofi
* flameshot
* nerd-fonts

## Installation

```bash
curl -fsSL https://raw.githubusercontent.com/kartavkun/i3-dotfiles-minimal/main/install.sh | sh
```
